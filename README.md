# Privacy Policy - Restaurant Orders

Este directorio contiene la política de privacidad de Restaurant Orders.

## Archivos

- `index.html` - Página web con la política de privacidad completa
- `terms.html` - Términos de Uso de la aplicación

## Hosting

Esta página debe ser alojada públicamente para cumplir con los requisitos de Google Play Store.

### Opción 1: GitHub Pages (Recomendado - Gratis)

1. Crear un repositorio público en GitHub
2. Subir este directorio
3. Activar GitHub Pages en Settings → Pages
4. La URL será: `https://[tu-usuario].github.io/[nombre-repo]/`

### Opción 2: Netlify (Gratis)

1. Crear cuenta en netlify.com
2. Arrastrar esta carpeta a Netlify Drop
3. Obtendrás una URL: `https://[nombre-aleatorio].netlify.app`

### Opción 3: Vercel (Gratis)

1. Crear cuenta en vercel.com
2. Importar este directorio
3. Deploy automático

## URL para Google Play Store

Una vez alojado, usa la URL pública en:
- Google Play Console → Política de privacidad
- app.json → privacy policy URL (opcional)
